# wedding_service

We are getting married, so we need a website to host our wedding invitations.

## Features

- Secure HTTPS hosting with self-signed certificates
- Guest invitation and RSVP management
- Event schedule and updates
- MySQL database for data persistence
- Prometheus metrics and Grafana monitoring
- Structured JSON logging with Go's slog
- Docker-based deployment with volume persistence

## Prerequisites

- [Go 1.24+](https://golang.org/doc/install)
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)
